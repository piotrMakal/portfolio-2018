Maya Wire Plugin

Year: 2017 - Present
Fields: 3D Modeling, Mathematics, Programming, Tools Development
Used: Autodesk Maya, Autodesk Maya API, Python
Description: Maya Wire Plugin enables fast and easy creation of wires, cables, pipes, ropes, ribbons and other wire-like shapes.

Why Bother?

Autodesk Maya has its own ways for wire creation, so why creating separate tool for this? In my opinion there are exactly 3 reasons: (1) it takes too much time to create a simple cable, (2) final result is hard to maintain/modify later on, (3) the algorithms Maya provides leave a lot to be desired.
I was especially affected by this when working on a game, where I had to model a tons of cables. That's where the original idea came to my mind.

License: CC BY-SA 4.0

With plugin comes a new node (wireMeshCreator) that is responsible for all the calculations and outputing correct wire mesh data. This allows modifications to geometry at any point in time, even after saving and reopening Maya scene. Attributes of the node can be mixed together giving user the flexebility of achiving different shapes, from real-world: cables, pipes, ribbon and ropes, to more abstract ones.
